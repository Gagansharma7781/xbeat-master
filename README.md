## X-Beat | React eCommerce Web App

## Features:

- #### Add Product to the Cart
- #### Remove Product from the Cart
- #### Increment / Decrement the Product's Quantity
- #### Product's Total Price / Discount Calculations
- #### Custom Hooks
- #### Fully Responsive
- #### Dark Theme
- #### Multi Pages

## Technologies used:

- #### **React JS**
- #### **React Router**
- #### **React Swiper**
- #### **SASS**


## Available Scripts

#### `npm start`

#### `npm test`

#### `npm run build`

#### `npm run eject`
